# ⚽ FIFA World Cup Data Analysis using SQL (Sky Sports Project)

## 📌 Problem Statement

Analyze FIFA World Cup team performance data to uncover insights related to team performance, scoring patterns, player usage, and overall match statistics.

---

## 🎯 Objective

To analyze football data using SQL and identify key trends in team performance, goal scoring, possession, and match outcomes across different groups.

---

## 📂 Dataset Description

This project uses two datasets:

### 1️⃣ Group Stage Team Stats

Contains:

* Team name, group, rank
* Matches played, wins, losses, draws
* Goals scored and conceded
* Points and goal difference
* Expected goals metrics

### 2️⃣ Overall World Cup Stats

Contains:

* Player statistics (players used, avg age)
* Performance metrics (goals, assists, shots)
* Passing, possession, and defensive stats
* Advanced metrics like xG, shots per 90, etc.

---

## 🛠 SQL Concepts Used

* SELECT, WHERE, DISTINCT
* Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
* GROUP BY & ORDER BY
* Subqueries
* JOIN operations
* Filtering & conditional logic

---

## 🔍 Key Analysis Performed

### ⚽ Team Performance Analysis

* Total teams and group-wise distribution
* Teams with highest and lowest goals
* Win, loss, and draw percentages

### 📊 Match & Goal Analysis

* Goals scored vs conceded
* Goal difference trends
* Expected goals vs actual goals

### 🏆 Group Analysis

* Number of teams in each group
* Average wins per group
* Group-wise performance comparison

### 📈 Player & Team Stats

* Teams with highest players used
* Average age of teams
* Possession and passing analysis

### 🔗 Advanced Analysis (Joins)

* Group-wise average age
* Total fouls per group
* Total games played per group

---

## 💡 Key Business Insights

* 🏆 Teams with higher goal difference tend to dominate group rankings
* ⚽ Expected goals (xG) provides deeper insight than actual goals scored
* 🌍 Some groups are more competitive with closely matched teams
* 📊 Teams with higher possession do not always win matches
* 👥 Teams using more players show better squad depth and flexibility
* 📈 Strong attacking teams also show higher assists and shot accuracy

---

## 🚀 Conclusion

This project demonstrates how SQL can be used to analyze sports data and extract meaningful insights about team performance, match outcomes, and player statistics.
